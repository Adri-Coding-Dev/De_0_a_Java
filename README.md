## 🎯 Objetivo del Proyecto

Brindar una formación **completa, progresiva y práctica** en programación con Java, guiando al estudiante desde los conceptos más básicos hasta el desarrollo de proyectos reales, aplicando **buenas prácticas**, **patrones de diseño** y metodologías profesionales.

## 🌟 Características Principales

✅ **Desde cero absoluto** - Sin requisitos previos de programación
✅ **Enfoque práctico** - Ejercicios y proyectos en cada módulo
✅ **Código fuente organizado** - Estructura clara por temas
✅ **Actualizado** - Java 21+ y mejores prácticas modernas
✅ **Soporte continuo** - Nuevos contenidos periódicos
✅ **Comunidad activa** - Espacio para dudas y colaboración

## 📚 ¿Qué Aprenderás?

### 🔰 **Fundamentos**
- Introducción a Java y entorno de desarrollo
- Variables, tipos de datos y operadores
- Estructuras de control (if, switch, bucles)
- Métodos y funciones
- Arrays y colecciones
- Integradores
- Colecciones de datos

## 🚀 Empezando

Sigue estos pasos para comenzar tu aprendizaje:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/Adri-Coding-Dev/De_0_a_Java.git
   cd De_0_a_Java
   ```

2. **Configura tu entorno:**
   - Instala [JDK 17 o superior](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)
   - Elige un IDE: [IntelliJ IDEA](https://www.jetbrains.com/idea/), [Eclipse](https://www.eclipse.org/) o [VS Code](https://code.visualstudio.com/)

3. **Sigue el orden de los vídeos:**
   - Comienza por el módulo 01
   - Mira el vídeo correspondiente
   - Practica con los ejemplos del repositorio
